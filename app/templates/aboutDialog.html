SimplyWrite is a free web distraction writing tool for those who like write.

SimplyWrite recognizes the lightweight markup language Markdown, an easy-to-read, easy-to-write plain format which allow to enrich the text.

Note SimplyWrite stores all your writes on the client side. It makes use of the HTML5 local storage features so no server is required.
