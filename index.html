SimplyWrite is a simple free web distraction writing utility as a web clone of WriteRoom, OmmWriter, FocusWritter and many others.

SimplyWrite is a simple tool I (@acanimal) used to test IndexedDB features of HTML5, so you need a HTML5 compliant browser.

Because the use of IndexedDB HTML5 feature all the content is stored locally on your machine, no need to send your writes accros the web or store it in "strange" servers.

SimplyWrite is written by acanimal as a free web distraction writing tool. Special thanks to css3buttons, html5boilerplate and jQuery, jQueryUI projects.
